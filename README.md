# FIAP Landing Page

Uma aplicação Next.js moderna com TypeScript e Tailwind CSS.

## 🚀 Tecnologias

- **Next.js 15** - Framework React para produção
- **TypeScript** - Tipagem estática
- **SASS** - Framework CSS utilitário
- **React 19** - Biblioteca UI

# Como rodar o projeto (Next.js) e o Storybook — guia rápido

Este documento descreve os passos para rodar localmente o Next.js (app/pages) e o Storybook do repositório, além de comandos úteis para build, análise e troubleshooting.

---

## Requisitos

- Node.js LTS (recomendado: 18.x ou superior)
- npm ou yarn (ex.: npm >= 8)
- Git (para clonar o repositório)

Verifique versão do Node:

````bash
node -v
npm -v
## 🛠️ Instalação

```bash
# Instalar dependências
npm install

# Executar em desenvolvimento
npm run dev

# Executar storybook
npm run storybook
````

## 📝 Scripts Disponíveis

- `npm run dev` - Inicia o servidor de desenvolvimento
- `npm run build` - Cria build de produção
- `npm start` - Inicia servidor de produção
- `npm run lint` - Executa o linter

## 🌐 Acessar

Abra [http://localhost:3000](http://localhost:3000) no navegador.
